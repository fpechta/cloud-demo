EDEKA Service Monitor

Cloud-basierte Überwachung der EDEKA-Dienste

Funktionsweise: Diese Anwendung überwacht die Verfügbarkeit verschiedener EDEKA-Services in Echtzeit und loggt alle Prüfungen in Azure Blob Storage. Entwickelt als Demo für Cloud-Infrastructure-Monitoring.

Tech Stack:

Python Flask Azure Kubernetes (AKS) Azure Blob Storage Docker Terraform
